ESSENTIAL STEPS
================================================================================

1. INSTALL NODE.JS LOCALLY
   - Download from: https://nodejs.org/ (v18 or higher)
   - Verify installation:
     node --version
     npm --version

2. CREATE NEXT.JS PROJECT
   - Navigate to: C:\Users\VanRo\Documents\Claude.AI\IGNTWebsite\
   - Run this command:
     npx create-next-app@latest ignt --typescript --tailwind --eslint --app --src-dir --import-alias "@/*" --no-git

3. DROP IN GENERATED CODE
   - Copy all generated files from Phases 2-9 into the new project
   - Organize by directory structure (src/components/, src/lib/, src/app/, etc.)
   - Copy BRANDING_GUIDE.md to project root

4. INSTALL DEPENDENCIES
   - npm install resend @resend/react-email
   - npm install @supabase/supabase-js
   - npm install framer-motion zod
   - npm install -D shadcn-ui

5. SET UP ENVIRONMENT
   - Create .env.local file with:
     NEXT_PUBLIC_SUPABASE_URL=<your_supabase_url>
     NEXT_PUBLIC_SUPABASE_ANON_KEY=<your_key>
     SUPABASE_SERVICE_ROLE_KEY=<your_key>
     RESEND_API_KEY=<your_key>
     UPSTASH_REDIS_REST_URL=<your_url>
     UPSTASH_REDIS_REST_TOKEN=<your_token>

6. SET UP SUPABASE
   - Create account at supabase.com
   - Run SQL schema from SUPABASE_SETUP.md
   - Create cv-uploads storage bucket with appropriate permissions

7. TEST LOCALLY
   - npm run dev
   - Open http://localhost:3000

8. DEPLOY TO VERCEL
   - Push code to GitHub
   - Import repository in Vercel
   - Add environment variables
   - Deploy
